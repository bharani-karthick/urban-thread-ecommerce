# 👗 urban-thread-ecommerce

A modern, fully responsive fashion e‑commerce website built with HTML, CSS, and JavaScript. Urban Thread delivers a clean and intuitive shopping experience across desktop, tablet, and mobile, featuring a fast and functional product search bar.

🔗 Live Demo: https://urbanthreadecommerce.netlify.app/

📦 Repository: https://github.com/bharani-karthick/urban-thread-ecommerce

# ✨ Overview

Urban Thread is a frontend fashion e‑commerce project focused on responsive design, clean UI, and real‑world shopping flows. The project was designed to simulate a professional online fashion store while keeping the codebase simple, lightweight, and easy to understand.

This project highlights strong fundamentals in HTML structure, CSS responsiveness, and JavaScript‑driven interactivity without using any frameworks.

# 🛠️ Technologies Used

- HTML5 – Semantic and accessible structure

- CSS3 – Flexbox, Grid, media queries, responsive layouts

- JavaScript (ES6+) – Search functionality and UI interactions

- Responsive Web Design – Desktop, Tablet, and Mobile support

# 🚀 Features

Key features of Urban Thread:

-  Fashion E‑commerce UI – Clean and modern store layout

-  Fully Responsive Design – Optimized for all screen sizes

-  Working Search Bar – Instantly search products using JavaScript

-  Smooth UI Interactions – Fast and user‑friendly navigation

-  Modern Styling – Minimal, trendy fashion aesthetics

# 📱 Responsive Behavior

The website adapts seamlessly to:

-  Desktop screens

-  Tablets

-  Mobile devices

Layouts are built using CSS Grid, Flexbox, and media queries to maintain consistency and usability across devices.

# ⌨️ Key Functionality

- Product Search: JavaScript‑powered search bar for filtering products

- Responsive Navigation: Mobile‑friendly layout adjustments

- Clean Layout Structure: Easy‑to‑maintain and readable code

# 🧠 Development Process

- Planned the layout and user flow for a fashion e‑commerce website

- Built responsive layouts using CSS Grid and Flexbox

- Styled the UI with a modern fashion‑focused design approach

- Implemented JavaScript logic for search functionality

- Tested responsiveness on desktop, tablet, and mobile screens

- Refined UI spacing, alignment, and interactions

# 📚 What I Learned

Through building Urban Thread, I learned:

-  Best practices for responsive web design

-  Structuring real‑world e‑commerce layouts

-  Implementing search functionality using vanilla JavaScript

-  Improving UI/UX for fashion‑focused websites

-  Writing clean, maintainable front‑end code
